# 🎓 Student Result Management System

A command-line based Python project to manage and display student academic results with automatic grade calculation.

---

## 📌 About the Project

This project was built as part of my Python learning journey during my final year of B.Tech CSE at TKR College of Engineering & Technology, Hyderabad.

It allows users to add students, enter subject marks, automatically calculate grades and percentage, and view individual or class-wide results.

---

## ✨ Features

- ➕ Add new student with marks for 5 subjects
- 📊 View individual student result with grade and pass/fail status
- 👥 View all students with percentage and grade
- ✏️ Update marks for any student
- 🗑️ Delete a student record
- 🏆 View Top 3 ranked students
- 🚪 Clean exit option

---

## 🛠️ Technologies Used

- **Language:** Python 3
- **Concepts:** Functions, Dictionaries, Loops, Conditionals, Input Validation

---

## ▶️ How to Run

1. Make sure Python 3 is installed on your system
2. Clone this repository:
   ```bash
   git clone https://github.com/anushavadyala/student-result-management-system.git
   ```
3. Navigate to the project folder:
   ```bash
   cd student-result-management-system
   ```
4. Run the program:
   ```bash
   python student_result_system.py
   ```

---

## 📊 Grading System

| Percentage | Grade | Result |
|------------|-------|--------|
| 75% and above | A | PASS |
| 60% - 74% | B | PASS |
| 50% - 59% | C | PASS |
| 35% - 49% | D | PASS |
| Below 35% | F | FAIL |
